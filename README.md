# SortGarbageRussia
App that helps sorting garbage

Инструменты
-----------
* Kotlin
  * Android Extensions
  * Coroutines
  * DateTime (to be deleted)
* ObjectBox
* Retrofit
  * GSON converter
* GSON
* CameraX
* MLKit
  * Barcode Scanning
  * CameraX extensions
* Zxing
* JUnit5
